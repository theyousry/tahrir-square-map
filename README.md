# Tahrir Square Map Project

## Project Overview

I've developed a project of _a single-page web application_ featuring a map of **El-Tahrir Square** and various points of interest and i was happily adding functionality to this map including highlighted locations and third-party data about those locations that help users browse all included landmarks **using [Knockout](http://knockoutjs.com/) Framework** plus any selected additional information about a landmark is gonna be **powered by [Google Maps API](https://developers.google.com/maps/documentation/javascript/) & [Foursquare API](https://developer.foursquare.com/docs/)**.

## Running the App:
##### Clone or Download this Repository

* install all project dependencies with `npm install`
* start the development server with `npm start`
